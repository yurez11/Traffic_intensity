# Интенсивность движения (Traffic Intensity)

Программа для анализа и прогнозирования интенсивности автомобильного движения с использованием метода линейной регрессии.

## Основные возможности

- Ввод и редактирование данных по годам и интенсивности движения (авт./сут.)
- Автоматическая нумерация строк и заполнение года по порядку
- Проверка корректности вводимых данных (год: 1900–2100, интенсивность ≥ 0)
- Расчёт линейной регрессии:  
  - Коэффициенты уравнения \( N = a + b \cdot t \), где \( t = \text{год} - 1980 \)  
  - Средняя интенсивность и среднее квадратическое отклонение (СКО)  
  - Коэффициент корреляции  
  - Ошибка аппроксимации (%)  
  - Прогноз на 2000 год
- Отображение результатов:
  - В виде таблицы с теоретическими значениями, отклонениями и относительной ошибкой
  - На графике (исходные данные + линия регрессии + точка прогноза)
- Сохранение:
  - Исходных данных в собственном формате `.t_i_k`
  - Результатов расчётов в текстовый файл (`.txt`)
  - Графика в изображение (PNG, JPG, BMP, EMF)
- Удобный интерфейс:
  - Горячие клавиши (Ctrl+N, Ctrl+O, Ctrl+S, F5 и др.)
  - Копирование/вставка/вырезание ячеек (Ctrl+C/V/X)
  - Перетаскивание строк мышью
  - Вставка новой строки в текущую позицию
  - Удаление одной или нескольких строк с подтверждением

## Технологии

- Язык: C++ (Embarcadero C++ Builder)
- Библиотеки: VCL, TeeChart
- Формат данных: бинарный (`.t_i_k`) — хранит год (int16) и интенсивность (uint32 × 10 для дробной части)

## Структура проекта

- `trafic_intensity_code.h` — заголовочный файл с описанием класса формы
- `trafic_intensity_code.cpp` — основная логика приложения
- `*.dfm` — форма интерфейса (генерируется IDE)

## Как использовать

1. Скомпилируйте проект в **Embarcadero C++ Builder** (версия, поддерживающая VCL и TeeChart)
2. Запустите исполняемый файл
3. Добавьте данные через меню **Файл → Новый**
4. Нажмите **Расчёт → Выполнить расчёт** (или F5)
5. Сохраните результаты или график при необходимости

> Проект не предназначен для сборки в Visual Studio, Qt или других средах — только C++ Builder с компонентами VCL.
